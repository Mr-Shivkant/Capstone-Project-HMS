# 🏨 Hotel Management System (HMS)

A **web-based Hotel Management System** developed as a **Capstone Project** to simplify hotel operations such as room booking, customer management, billing, and staff handling.

## 📌 Project Overview

Managing hotel operations manually can lead to:

- Data entry errors
- Time-consuming processes
- Booking conflicts
- Poor record management

This project provides a **centralized digital solution** that helps hotel administrators manage all operations efficiently through a user-friendly interface.

---

## 🎯 Objectives

- Automate hotel room booking
- Manage customer records
- Track room availability
- Handle billing details
- Improve operational efficiency
- Reduce manual paperwork

---

## 🚀 Features

### 🔐 Authentication
- Admin login system
- Secure access to dashboard

### 🛏 Room Management
- Add rooms
- Update room details
- Delete rooms
- Track availability

### 📅 Booking Management
- Create new bookings
- Modify bookings
- Cancel bookings
- Check-in / Check-out support

### 👤 Customer Management
- Store guest details
- View booking history
- Maintain records

### 📊 Dashboard
- Overview of:
  - Total rooms
  - Booked rooms
  - Available rooms
  - Customers

### 👨‍💼 Staff Management
- Maintain staff details
- Role-based management
